# Sistem Informasi Akademik (SIAKAD) - PHP Native

Proyek ini adalah aplikasi Sistem Informasi Akademik yang dibangun menggunakan PHP Native (tanpa framework).

## Struktur Proyek

- `/config` - Konfigurasi aplikasi seperti koneksi database.
- `/database` - File SQL untuk skema database.
- `/assets` - CSS, JS, dan gambar.
- `/includes` - Komponen yang dapat digunakan kembali seperti header dan footer.
- `/pages` - Halaman-halaman fitur aplikasi.
